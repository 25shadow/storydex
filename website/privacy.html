隐私说明

生效日期：2026 年 7 月

1. 数据本地化原则

Storydex 是一款本地优先的桌面写作工作台。我们坚守以下核心原则：

  • 我们不会读取您的任何创作内容。您的所有小说正文、角色设定、WIKI 条目、变量记录、创作预设等项目文件均存储在您的本地设备上，Storydex 不会扫描、读取、收集或上传这些内容。
  • 我们不会将您的数据上传至云端。本软件不包含任何自动上传用户内容至远程服务器的功能。您的所有数据始终由您本地持有和控制。
  • 我们没有后台数据收集。本软件不嵌入任何用户行为分析、埋点统计或数据追踪 SDK。

2. 软件更新时涉及的连接

本软件提供应用内更新功能。检查更新时，软件会向更新服务器发送以下信息：

  • 当前版本号（用于判断是否有新版本）
  • 操作系统平台标识（Windows）

更新过程不会传输您的任何个人身份信息、项目文件路径或创作内容。

3. AI 模型调用

Coomi Agent 需要调用您自行配置的第三方 AI 模型服务（如 OpenAI、DeepSeek 等）。相关说明：

  • 模型请求由本软件直接向您配置的模型供应商服务器发起。
  • Storydex 不代理、不中转、不记录任何模型请求的内容。
  • 您在模型请求中发送的内容（如章节文本、角色设定等）受您与对应模型供应商之间隐私政策的约束。
  • 您的 API Key 存储在本地配置文件中，不会发送给 TensorHub 或任何第三方。

4. 版本控制 (Git)

本软件内置 MinGit，用于为您的写作项目提供本地版本控制。Git 仓库仅存在于您的本地项目目录中，不会自动推送至任何远程仓库。

5. 日志与诊断

本软件可能在本地生成运行日志，用于排查程序错误。这些日志文件存储在软件安装目录或用户配置目录中，不会自动上传。我们仅在您主动提供日志文件以协助排查问题时才会接触这些数据。

6. 第三方服务

本软件可能包含指向外部网站（如 GitHub）的链接。点击这些链接后，您的浏览器行为受对应网站的隐私政策约束，Storydex 不对此负责。

7. 儿童隐私

本软件不面向 13 岁以下儿童。我们不会故意收集儿童的个人信息。

8. 隐私说明的变更

我们可能会不时更新本隐私说明。更新后的版本将在软件新版本中发布，继续使用即视为接受更新后的说明。

9. 联系方式

如对本隐私说明有任何疑问，可通过 GitHub Issues 联系我们：TensorHub-ORG/Storydex
