# Storydex v0.3.4

## 桌面差分更新热修复

- 修复 v0.3.3 差分安装完成时强制立即拉起程序，可能在应用文件仍处于替换窗口时触发 `Cannot find module ... electron-updater/out/main.js` 主进程错误的问题。
- 安装更新后不再强制抢跑新进程；静默安装完成后由用户重新打开 Storydex，避免读取到半更新状态的运行目录。
- 自动更新组件遇到临时模块加载失败时会按确定性间隔重试，文件就位后自动恢复更新能力，不再永久停留在“缺少 electron-updater 依赖”状态。
- “更新与关于”页面不再重复显示同一条不支持更新错误，并明确显示更新组件初始化状态。
- 打包资源检查新增 `electron-updater/out/main.js` 强制校验；Electron E2E 新增安装切换窗口模拟，验证入口文件短暂缺失后能够自动恢复。

## 更新方式

已安装 v0.3.3 的用户可以检查并下载 v0.3.4。安装开始后 Storydex 会退出，安装完成后请手动重新打开应用。也可以下载完整安装包覆盖安装，项目文件、会话记录和用户配置不会随应用安装目录替换而删除。

安装包与便携包继续内置可迁移 Python 3.9 环境、后端依赖和 MinGit，无需额外安装 Python 或 Git。
