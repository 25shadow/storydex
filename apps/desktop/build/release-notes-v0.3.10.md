# Storydex v0.3.10

本次为长会话与发布稳定性更新。

- 按 Storydex 项目与 sessionId 隔离 Agent 运行态、意图状态和会话恢复，避免跨工作区串用上下文。
- 完善 ContextTrace：记录上下文来源、候选与实际纳入量、截断、重复段落、Provider 请求及 token 估算，便于定位长会话问题。
- 增强 LLM 录制/回放一致性，规范工具参数、稳定路径与累计 usage 快照，并在请求结束后回收请求级指标。
- 修复重复工具开始状态；继续保证需复核的故事知识变更不写入，混合批次只持久化已接受操作。
- 修复本地 Release 版本漂移、编码校验误扫本地输出和前端回归测试先于生产构建执行的问题。

已安装用户可在 Storydex 内检查更新；便携版请下载新版本重新解压。
